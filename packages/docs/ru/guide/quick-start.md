# Быстрый старт

Перед началом потребуется некоторые базовые настройки:

1. **Добавить настройки подключения.** `Settings -> Proxies -> Create`. Если у тебя есть прямой ключ, можно просто указать его в соответствующее поле. В противном случае, используем настройки прокси   

![](/quick-start-settings.jpg){width=300px}

2. **Указать настройки прокси для промптов**. 

![](/quick-start-flow-1.jpg){width=300px}

Надо указать прокси для каждого промпта

![](/quick-start-flow-2.jpg){width=300px}

В конце не забудь сохранить (каждый промпт и весь флоу)

![](/quick-start-flow-3.jpg){width=300px}

3. **Добавить карточки персонажей.** Для начала тебе нужны карточки персонажа и персоны (подробнее смотри [в разделе](/ru/features/characters.md)). Можно импортировать любую v2 карточку.

![](/quick-start-characters.jpg){width=300px}

4. **Создать чат.**

![](/quick-start-chats.jpg){width=300px}

- Выбрать карточку персонажа и персоны
- Отметить активную персону
- Выбрать флоу

![](/quick-start-chat.jpg){width=300px}

5. Готово, вы восхитительны! (Не забудь подправить гритинги под безжоп)

![](/quick-start-done.png){width=300px}
