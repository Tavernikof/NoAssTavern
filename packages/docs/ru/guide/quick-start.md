# Быстрый старт

Перед началом потребуется некоторые базовые настройки:
1. Добавить настройки подключения.
   ![](https://i.postimg.cc/j5h5J0Cf/Frame-1-1.jpg)

2. Добавить карточки персонажей и персоны
   Персоны больше не нужны. Вместо этого создавай свою карточку прямо тут.
   !!!info Для карточек, которые будут использоваться как персоны, в описании все равно надо указывать `{{char}}`
   ![](https://i.postimg.cc/ThMPWdfT/Frame-2-4.jpg)

3. Создать чат

![](https://i.postimg.cc/brKKyd89/Frame-3-4.jpg)

4. Готово, вы восхитительны!
