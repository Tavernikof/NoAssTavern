# Быстрый старт

Перед началом потребуется некоторые базовые настройки:

1. **Добавить настройки подключения.** `Settings -> Proxies -> Create`. Если у тебя есть прямой ключ, можно просто указать его в соответствующее поле. В противном случае, используем настройки прокси   

![](/quick-start-settings.jpg){width=300px}

2. **Добавить карточки персонажей.** Для начала тебе нужны карточки персонажа и персоны (подробнее смотри [в разделе](/ru/features/characters.md)). Можно импортировать любую v2 карточку.

![](/quick-start-characters.jpg){width=300px}

3. **Создать чат.**

![](/quick-start-chats.jpg){width=300px}

- Выбрать карточку персонажа и персоны
- Отметить активную персону
- Выбрать флоу

![](/quick-start-chat.jpg){width=300px}

4. Готово, вы восхитительны!

![](/quick-start-done.png){width=300px}
