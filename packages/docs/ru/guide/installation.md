# Установка и запуск

Для запуска существует 2 пути:

## Облачный 
::: tip Клик
https://tavernikof.github.io/NoAssTavern/
:::
Простой способ потыкать функционал без заморочек с установкой. В этом режиме все данные хранятся в браузере.

В любой момент можно скачать полный дамп и перенести его в локальную версию или подключить локальный бекенд прямо к облачной версии.

## Локальный
Для этого тебе понадобятся:
1. [Git](https://git-scm.com/)
2. [Node.js 20+](https://nodejs.org/en)

Установка и запуск:
1. Скачивание проекта
```bash
git clone git@github.com:Tavernikof/NoAssTavern.git
```

Или скачать последнюю версию с https://github.com/Tavernikof/NoAssTavern/releases (Ворнинг: Потом замонаешься с обновлениями, лучше через гит)

2. В папке с проектом запустить `start.bat` (шиндоус) или `start.sh` (линукс/мак)

3. Открыть http://localhost:4501 (Может быть другой порт, правильная ссылка будет в консоли)
