# Сообщения

Сообщения — это отдельные обмены репликами в ваших чатах.

## Типы сообщений

NoassTavern обрабатывает разные роли сообщений:

| Роль | Описание |
|------|-------------|
| **User** | Сообщения, которые вы отправляете |
| **Assistant** | Ответы AI-персонажа |
| **System** | Внутренние сообщения (обычно скрыты) |

## Действия с сообщениями

### Редактирование

Дважды щёлкните любое сообщение для редактирования. Это полезно для:

- Исправления опечаток в ваших сообщениях
- Корректировки ответов ИИ для соответствия вашей истории
- Исправления поведения персонажа

### Перегенерация

Нажмите кнопку перегенерации для получения нового ответа ИИ:

- Текущий ответ становится «свайпом»
- Генерируется новый ответ
- Полезно, когда ИИ неправильно понял или дал плохой ответ

### Свайпы (альтернативные ответы)

При перегенерации сообщения предыдущие ответы сохраняются как свайпы:

- Используйте стрелки влево/вправо для просмотра свайпов
- Каждый свайп — это другая версия того же ответа
- Можно переключаться между свайпами в любое время
- Помогает найти идеальный ответ

### Перевод

Переведите любое сообщение на другой язык:

- Использует настроенного AI-провайдера
- Перевод отображается рядом с оригинальным текстом
- Полезно для многоязычной ролевой игры или изучения языков

### Удаление

Удаление сообщений из чата:

- Удаление отдельных сообщений
- Беседа корректируется соответственно
- Будьте осторожны — удаление необратимо

### Просмотр деталей запроса

Просмотр технической информации о генерации сообщения:

- Полный промпт, отправленный ИИ
- Количество токенов (вход/выход)
- Использованная модель и провайдер
- Полезно для отладки или оптимизации

## Форматирование сообщений

Сообщения поддерживают базовое форматирование:

- **Жирный** текст с `**жирный**`
- *Курсив* с `*курсив*`
- Обычный текст параграфа

## Внешние блоки

К сообщениям могут быть прикреплены внешние блоки:

- Дополнительный контент, не отображаемый в основном сообщении
- Используется потоками и блоками кода
- Можно просмотреть в деталях сообщения

## Обработка ошибок

При неудачной генерации сообщения:

- На сообщении появляется индикатор ошибки
- Можно просмотреть детали ошибки
- Перегенерируйте для повторной попытки
- Проверьте API-ключ и настройки подключения
