# Режим ассистента

![](/assistant.png){width=300px}

Отдельный режим, который позволяет делать запросы в режиме чата. Работает аналогично [Open WebUI](https://github.com/open-webui/open-webui), [LibreChat](https://www.librechat.ai/) и прочим фронтендам.
Особенности:
- **Умеет работать со всеми API** + поддержка прокси. Главная фича, потому что ни один из существующих фронтендов нормально не поддерживает работу с OpenAi, Gemini и Claude, либо не поддерживает прокси.
- Свайпы
- Генерация картинок через Nano Banana (только на Gemini API)
